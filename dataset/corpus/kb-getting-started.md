# Getting started with Northwind Cloud

Northwind Cloud is a hosted workspace for small teams. To get started, create an organisation, invite teammates from Settings > Members, and connect your first data source. New organisations start on the Free plan, which supports up to three members and one project. You can upgrade at any time from Settings > Billing.
